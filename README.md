# Church Website

A modern church website built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- Responsive design with mobile-first approach
- Custom color palette (navy, gold, neutral)
- Inter font for modern look
- Pages: Home, About, Ministries, Sermons, Events, Give, Prayer, Contact
- SEO optimized with meta tags, sitemap, robots.txt
- Sticky navbar with scroll effect
- Accessibility improvements (aria labels, focus states)
- Smooth hover animations and transitions
- Ready for deployment on Vercel

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser
5. Build for production: `npm run build`

## Project Structure

- `src/app/` - Next.js App Router pages
- `src/components/` - Reusable React components (Navbar, Footer, Hero, Card, Button, Form)
- `public/` - Static assets (images, sitemap, robots.txt)

## Deployment

1. Push the code to a GitHub repository
2. Connect the repository to Vercel
3. Deploy automatically or manually

## Technologies Used

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React

## Notes

- Placeholder images and content are used throughout
